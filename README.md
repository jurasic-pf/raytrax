# Raytrax: an ECRH ray tracer based on JAX

Raytrax is a ray tracer for electron cyclotron resonance heating based on JAX.

The physics approach is based on the approach used for the TRAVIS Fortran ray tracer, see [*Marushchenko, Turkin, and Maassberg, Comput. Phys. Commun., Vol. 185, Iss. 1, 2014, pp. 165-176*](https://doi.org/10.1016/j.cpc.2013.09.002).
